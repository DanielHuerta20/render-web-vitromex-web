# Render Web 
> Version web de render vitromex

## Develop
Para modo desarrollador primero necesitas instalar las dependecias

```bash
npm i
```

Levantar servidor
```bash
npm run start
```

## Build
Para generar el build con 


```bash
npm run build
```

pudes probarlo con 


```bash
npm run serve
```
